AngularJS Modal Date and Time Pickers

Android inspired simple date and time pickers for desktop and mobile. AngularJS directive goodness. No jQuery, storing values on DOM elements, events etc.

Demo

Resize your browser to see modal switch for small screens.

{{myDateTime | dateFormat}} {{myDateTime | timeFormat}}

Download: ZIP (16KB)

Info

Built with excellent Moment.js library. Modals based on CSS Modal. Touch detection by Modernizr.

Browser support: Chrome, Firefox, Safari, iOS, Android. Microsoft? Less than IE10 you'll have some work to do!

For i18n I recommend using an Angular service to load the locale language bundle and switch in the corresponding Moment.js language. Apply the localized strings using an Angular filter. Take a look at ngTranslate.
